# Images Directory

Every liked item MUST have its image saved here alongside the JSON metadata.

## Naming
- `item-NNN.jpg` or `item-NNN.png` — matches the JSON filename number
- Example: `item-001.json` → `images/item-001.jpg`

## Why
- URLs rot and product pages disappear — saved images don't
- Needed for mood boards, content planning, and outfit recreation
- Essential as reference for AI image generation (brand shoots, try-on content)
- Gives Lana a visual library of her taste, not just metadata

## Backfill
Existing saves without images can be backfilled by revisiting their URLs and capturing images with matching filenames.
